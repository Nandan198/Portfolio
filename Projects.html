Project Title: Automated The process Of shopping On Amazon (Until-Payment)

Description of Project: In this project, I used Selenium WebDriver in Java to automate shopping on Amazon.in. The automation starts by opening the browser, navigating to Amazon.in, and sending search keywords to the search bar. It clicks on the desired product, switches to the new window, and proceeds to buy by providing the delivery address. This Java-based automation showcases my skills in test automation and web interaction, streamlining the shopping process for improved efficiency and user convenience on the Amazon platform.

Tools/Languages Used: Java, Selenium WebDriver, TestNG, JUnit
